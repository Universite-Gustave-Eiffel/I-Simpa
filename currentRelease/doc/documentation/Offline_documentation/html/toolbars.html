Toolbars¶

Toolbar ‘Project’¶

  • Create new project;
  • Open new project;
  • Save the project with the current filename;
  • Save the project with a new filename.

Toolbar ‘View/Camera’¶

  • 3D display:
    • 3D with faces (interior);
    • 3D without faces, all lines;
    • 3D with faces and all lines;
  • Camera:
    • Reinitialize camera;
    • Mode ‘Rotation/Zoom’;
    • Mode ‘First person’;

Toolbar ‘Meshing’¶

  • Generate and display the meshing (open a Dialog box for selecting the meshing parameters from a calculation code);
  • Remove the meshing from the 3D view;
  • Radio button ‘x’, ‘y, ‘z’: Select the planes X, Y, Z for displaying the meshing with the cursor;
  • The slider allows to move the meshing along the selected plane.

Toolbar ‘Selection’¶

  • Mode ‘Camera’: see menu ‘View’;
  • Mode ‘Face selection’: enable the face selection with the pointer (selected faces are colored on the 3D view and on the ‘Surfaces’ group on the tree ‘Data’ of the window ‘Project’). The procedure for the face selection is presented here.
  • Mode ‘Data extraction’: enable to extract a value on a colormap, with the pointer. The corresponding data is displayed in the window ‘Message’.

Toolbar ‘Simulation’¶

  • Reward to the last time step;
  • Play from the current time step;
  • Pause the animation;
  • Forward to the next time step;
  • Stop the animation (and return at the first time step);
  • Remove the animation (remove data animation from memory).
